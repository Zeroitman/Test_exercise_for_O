Страница описания работы программы

При первоначальном запуске программы Вас встречает главная страница с кнопкой "Получить случайного пользователя". По нажатию будет получен и сохранён в базу данных случайный пользователь, а Вас перенаправит на страницу просмотра профиля полученного пользователя.

На странице просмотра пользователя будут выведены все необходимые данные пользователя, в числе которых будет автоматически сгенерированная электронная почта и разделённые на числа и буквы хэши. В случае, если данные приходят не на латинице они будут переводиться на английский язык модулем yandex-translater, так как его алфавит полностью соответствует латинскому алфавиту. Однако, полагаю, что данный модуль вполне может не распознать некоторые символы. Все полученные данные будут спрятаны под кнопкой "Все данные пользователя".

Также вы можете посмотреть всех полученных пользователей кликнув на ссылку "Полученные пользователи" в навбаре

API
После получения пользователей перезагрузите сервер для корректного отображения данных в API Получить доступ к API можете по адресу "../hostname/user/", где будут выведены все данные случайного пользователя из тех что имеются в базе.

По адресу "../hostname/short/" вы увидете краткую информацию случайного пользователя из тех что имеются в базе.

По адресу "../hostname/numbers/" вы увидете только цифры хешей случайного пользователя из тех что имеются в базе.
